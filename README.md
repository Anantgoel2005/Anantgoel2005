### 🚀 What I've Built

Here are some of my standout projects:

#### 🚦 PathWise & Traffic Intelligence Dashboard
A high-performance, real-time AI traffic monitoring system for urban environments.
- **Tech:** Ultralytics YOLO-World, ByteTrack, FastAPI, React, WebSocket
- **Features:** Translates monocular camera feeds into a Bird's-Eye View (BEV), calculates Time-to-Collision (TTC), detects violations/accidents, and uses a scikit-learn ML predictor for risk assessment.

#### 🎮 TETR.IO Bot (Computer Vision & AI)
An autonomous agent that plays TETR.IO using screen capture and a heuristic AI engine.
- **Tech:** Python, OpenCV, MSS, Pynput, Numpy
- **Features:** Uses the Dellacherie heuristic algorithm, 9-0 stacking strategy, and 1-piece lookahead to achieve high-speed Tetris line clearing.

#### 🫁 Pneumonia Detection API
A deep learning web application to detect pneumonia from chest X-rays.
- **Tech:** PyTorch, Flask
- **Features:** Complete end-to-end pipeline from model training to a deployable REST API.

#### 📈 Indian Stock Analyzer
A comprehensive stock analysis tool packaged as a desktop application.
- **Tech:** Python, PyInstaller
- **Features:** Automated build pipelines and distribution for easy user access.

#### 🛡️ Fog IDS (Intrusion Detection System)
An academic project simulating a Network Intrusion Detection System for Fog computing environments.
- **Tech:** MATLAB
- **Features:** Uses CUSUM logic to detect anomalies and attacks in clustered fog networks.
